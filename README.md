# FelloWay

Monorepo for the FelloWay event networking product.

## Layout

| Path | Description |
|------|-------------|
| `frontend/` | Flutter mobile client (`felloway_client`) |
| `backend/` | API server (placeholder) |
| `specs/` | Feature specs and contracts |

## Run the Flutter app

```bash
cd frontend
flutter pub get
flutter run
```

See [frontend/README.md](frontend/README.md) for runtime config (`--dart-define`), API modes, and development standards.
